Job Description

As a Software Developer I, you will be responsible for designing and developing high-performance, scalable web applications using React, Redux, and related technologies. You will work closely with cross-functional teams, including designers, product managers, and other developers, to deliver exceptional user experiences.

Responsibilities

  • Develop and maintain complex, responsive web applications using ReactJS, Redux, and related libraries.
  • Write clean, efficient, and well-documented code that adheres to best practices and coding standards.
  • Collaborate with designers and product managers to translate wireframes and mockups into functional user interfaces.
  • Optimize application performance, including code optimization, lazy loading, and caching.
  • Troubleshoot and debug issues in existing applications.
  • Stay up-to-date with the latest trends and best practices in ReactJS development.

Requirements

  • Bachelor's degree in Computer Science, Information Technology, or a related field.
  • 3+ years of experience in ReactJS development.
  • Proficient in JavaScript, HTML, and CSS.
  • Experience with Redux, React Router, and other popular React libraries and frameworks.
  • Strong problem-solving and analytical skills.
  • Excellent communication and teamwork skills.
  • Familiarity with agile development methodologies.
  • Experience with testing frameworks and tools (e.g., Jest, Enzyme, Cypress).
  • Knowledge of server-side technologies (e.g., Node.js, Express) is a plus.

  • Professional development opportunities .
